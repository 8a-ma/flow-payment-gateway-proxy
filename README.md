# flow-payment-gateway-proxy
flow-payment-gateway-proxy es un servicio intermediario desarrollado en Node.js diseñado para simplificar la integración con la pasarela de pagos Flow (Chile). Actúa como un puente centralizado que gestiona la creación de órdenes, el rastreo de estados en tiempo real mediante Redis y el procesamiento de notificaciones asíncronas vía Webhooks.
